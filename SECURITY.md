# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.0.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

1. **Do not** open a public issue.
2. Contact [@arkanzasfeziii](https://github.com/arkanzasfeziii) directly.
3. Include: description, reproduction steps, potential impact.

**Response:** Acknowledgment within 48 hours. Fix within 30 days.

## Scope

This policy covers vulnerabilities in BlackForge itself, not in systems
tested using BlackForge during authorized engagements.
